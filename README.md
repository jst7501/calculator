# 갤럭시 스타일 계산기 🧮

삼성 갤럭시(One UI) 기본 계산기를 그대로 본뜬 React 계산기 앱입니다.

## 특징

- **갤럭시 One UI 디자인** — 흰색 원형 버튼, 회색 연산자, 초록색 `=`, 빨간 `C`/`⌫`, 초록 커서
- **실시간 미리보기** — 식을 입력하는 동안 결과가 아래에 미리 표시됩니다
- **완전한 계산 기능** — `+ − × ÷ %`, 괄호, 소수점, 음수, 연산자 우선순위 지원
- **계산 기록** — 상단 기록 아이콘으로 지난 계산을 확인하고 다시 불러오기
- **키보드 입력** — 숫자/연산자/Enter(=)/Backspace/Esc(C) 지원
- **안전한 평가기** — `eval` 미사용, 직접 구현한 Shunting-yard 파서

## 실행

```bash
npm install
npm run dev      # 개발 서버
npm run build    # 프로덕션 빌드
npm run preview  # 빌드 결과 미리보기
```

## 구조

| 파일 | 설명 |
|------|------|
| `src/App.jsx` | 계산기 UI 및 입력 처리 |
| `src/calc.js` | 수식 토크나이저 + 평가기 + 결과 포맷터 |
| `src/index.css` | 갤럭시 스타일 디자인 |
